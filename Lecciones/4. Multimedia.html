Lección 4: Multimedia.

Durante esta épica lección aprenderemos a añadir fotos, videos, y demás elementos multimedia en html5. Es a partir de este punto donde html5 se empieza a poner interesante.

Primero vamos con lo más simple, poner una imagen. Para ello se usa la etiqueta img src="" alt="". Donde el atributo src es lo mismo que href con los enlaces. Se usa todo lo de rutas internas y externas. Se puede poner un width y un height, pero es recomendable hacerlo en CSS3.

Ejemplo de imagen.

Ahora vamos a por un video. La única diferencia con las fotos es que esta es¡tiqueta se cierra. "video src=""> /video". Además, para acceder a los controles del video hay que poner el atributo controls. Igual para audios.
